Approve Sale Orders Based on customers credit limit

ERPish.com

The main features of this free appliction is :

  • Set the credit limit for every customer
  • For Sale Agents to confirm a sale, the order is checked against the customer limit
  • If the customer is to exceed their credit limit, all sale orders have to be approved by a sale Manager
